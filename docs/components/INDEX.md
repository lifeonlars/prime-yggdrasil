## Decision Guide

**When to use Yggdrasil blocks vs PrimeReact:**

| Scenario | Use |
|----------|-----|
| Navigation tabs with extra controls | `NavigationTabs` (Yggdrasil) |
| Simple tabs | `TabView` (PrimeReact) |
| Form field with label + error | `FormField` (Yggdrasil) |
| Just an input | `InputText` (PrimeReact) |
| Card with Yggdrasil styling | `Card` (Yggdrasil) |
| Complex card with header/footer | `Card` (PrimeReact) |

**Rule**: Check Yggdrasil blocks first - they're pre-configured with correct tokens and patterns.

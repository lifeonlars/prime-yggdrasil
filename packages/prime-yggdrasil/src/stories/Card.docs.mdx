# Card

A foundational block component that provides a consistent container with PrimeReact surface styling.

## Usage

Cards are **Tier 1 (Global Design System)** blocks. Use them for content grouping with consistent elevation and spacing.

```tsx
import { Card } from 'prime-yggdrasil';

<Card>
  <h3 className="text-color font-bold mb-2">Title</h3>
  <p className="text-color-secondary mb-0">Content goes here</p>
</Card>
```

## Rules

- **Visual styling**: Managed by PrimeReact classes (`surface-card`, `shadow-2`, `border-round`)
- **Layout composition**: Use Yggdrasil utilities for internal layout (`flex`, `grid`, `gap-*`, etc.)
- **No ad-hoc styling**: Avoid custom margins or padding beyond the card's default `p-4`

## Guidance for LLMs

When composing UI with Cards:
1. Use Cards for distinct content sections
2. Use Yggdrasil utilities grid/flex for layout between cards (`grid`, `col-*`, `gap-*`)
3. Keep card content semantic and simple
4. For repeated patterns across multiple views, consider promoting to a new Block

## Usage

### Buttons

```tsx
<Button label="Primary" />
<Button label="Success" severity="success" />
<Button label="Info" severity="info" />
<Button label="Warning" severity="warning" />
<Button label="Danger" severity="danger" />
```

### Tags

```tsx
<Tag value="Primary" />
<Tag value="Success" severity="success" />
<Tag value="Info" severity="info" />
<Tag value="Warning" severity="warning" />
<Tag value="Danger" severity="danger" />
```

### Messages

```tsx
<Message severity="success" text="Success Message" />
<Message severity="info" text="Info Message" />
<Message severity="warn" text="Warning Message" />
<Message severity="error" text="Error Message" />
```

## Design Principles

The color system follows these principles from the Yggdrasil design system:

- **Accessibility**: All color combinations meet WCAG AA standards for contrast
- **Consistency**: Semantic colors are applied consistently across all components
- **Context**: Colors convey meaning (success, warning, danger, info)
- **Hierarchy**: Primary colors establish visual hierarchy and brand identity

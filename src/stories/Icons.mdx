# Icons

PrimeIcons is the default icon library for PrimeReact with over 300+ icons. Icons can be used with the `icon` prop on components like Button, or directly using the `<i>` element with the appropriate class.

## Usage

```tsx
// In Button component
<Button icon="pi pi-check" label="Confirm" />

// Icon only button
<Button icon="pi pi-times" />

// Direct usage
<i className="pi pi-check"></i>
```
